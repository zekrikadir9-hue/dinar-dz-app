# DinarDZ - Mining, Shop & Referrals App

منصتك الشاملة للتعدين والمتجر والإحالات

## المميزات
- التعدين - اربح عملات كل يوم
- متجر متكامل - هواتف، إكسسوارات، بطاقات هدايا
- نظام إحالة - اربح بنسبة 10% من كل إحالة
- سحوبات حصرية
- شحن الهاتف - جيزي، أوريدو، موبليس
- دفع الفواتير

## التشغيل
```bash
npm install
npm start
```
